# MOS-Management-Ourselves-Systems

　このシステムは、AIによってユーザーのタスクを集中管理し、適宜解答案を作成または直接対応することを目的とします。
　対象タスクはビジネスからプライベートまで多岐に渡り、ユーザーからのチャット形式での個別依頼はもちろんのこと、使用している様々なデバイス・サービスと連携して自律的にタスク化し管理していきます。

## 主要機能（予定）

### Phase1 メイン機能

- タスク管理AIエージェント
- チャットスペースにて、ユーザーからのタスク提供またはその類推
- タスク消化フォロー

### Phase2 外部連携A（通知関係）

- カレンダー同期（iOS,Google,オリジナル...）
- 通知機能（メール,Discord,LINE,slack,デスクトップ...）

### Phase3 プライベートアプリケーション化（UI,UX）

- ガントチャート管理
- ファイル,リンクなどアップロード可能
- LLM呼び出し可（常駐も選択可）
- ユーザー管理・認証設定
- デザイン変更可能
- カレンダーアプリ化も視野
- セキュリティ関連

### Phase4 外部連携B（認証関係）

- メールや各種SNSとの連携・情報取得
- ユーザーのスマホやウェアラブル端末から位置情報やバイタルデータを定期的に取得し、行動パターンなどを蓄積・予測（要認証）
- ユーザーに許可された動作（誤作動を起こしても問題が少ない,あるいは定例的・機械的な作業など）のAIによる自律実行

### phase5 秘書化

- 自律駆動を基本とし、ユーザーの行動思考を先読みすることで最適な補助を行う
- 日常タスクであればこちらからの指示はもはや不要であり、todoリストの作成やスケジュール管理を先行して実施

### Phase6 リリース

- 機能を制限しつつ任意のOSのアプリケーションとしてリリース
- 共同作業可
- セキュリティおよびエージェントの権限設定が鬼門
- 第三者からの攻撃や乗っ取り,誤作動は重大インシデントになりうる

### Phase EX Virtual Doppelgänger
- 自分の思考・行動原理を完璧にトレース+αし、電子上にもう一人の自分を構築する。
- 全てのオンライン上の手続きはこれが行い、自分の脳のリソースを創作活動や趣味に当てることができる
- 一度構築に至れば、それは一人にとどまらない
- それはタスク管理などにとどまらず、一種の知的生命体の誕生と同義
